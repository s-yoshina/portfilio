=Valid Parentheses=

A program that checks whether all parentheses that are opened are closed, or vice versa, in proper order 
within a string.

<Examples>

Example 1:

Input: s = "()"
Output: true

Example 2:

Input: s = "()[]{}"
Output: true

Example 3:

Input: s = "(]"
Output: false

Problem link: https://leetcode.com/problems/valid-parentheses/
