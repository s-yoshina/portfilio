=Add Two Numbers=

A program that adds two values stored in a singly linked list that has the digits stored in reversed order.
The added value is returned again in reversed order as a singly linked list.

<Examples>
Example 1:
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807.

Example 2:

Input: l1 = [0], l2 = [0]
Output: [0]

Example 3:

Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
