=Binary Tree Inorder Traversal=

A program that traverses a binary tree inorder and returns the values of each node
that the program passes as a list.

<Example>
Example 1:

Input: root = [1,null,2,3]
Output: [1,3,2]

Example 2:

Input: root = []
Output: []

Example 3:

Input: root = [1]
Output: [1]

Problem link: https://leetcode.com/problems/binary-tree-inorder-traversal/
