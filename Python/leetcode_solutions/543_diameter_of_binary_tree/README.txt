=Diameter of Binary Tree=

A program that finds the length of the longest path between nodes in a binary tree and
returns it as an integer.

<Examples>
Example 1:

Input: root = [1,2,3,4,5]
Output: 3
Explanation: 3 is the length of the path [4,2,1,3] or [5,2,1,3].

Example 2:

Input: root = [1,2]
Output: 1
