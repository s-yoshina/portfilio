=Binary Tree Preorder Traversal=

A program that does a preorder traversal of a binary tree and returns the stored
values of each node that the program passes as a list in the order that the program
traversed the tree.

<Examples>
Example 1:

Input: root = [1,null,2,3]
Output: [1,2,3]

Example 2:

Input: root = []
Output: []

Example 3:

Input: root = [1]
Output: [1]

Problem link: https://leetcode.com/problems/binary-tree-preorder-traversal/
