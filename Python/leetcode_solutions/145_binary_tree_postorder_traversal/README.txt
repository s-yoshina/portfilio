=Binary Tree Postorder Traversal=

A program that does a postorder traversal of a binary tree where the program
returns the values stored in each node of the tree in the order of which the program
traverses the tree as a list.

<Example>
Example 1:

Input: root = [1,null,2,3]
Output: [3,2,1]

Example 2:

Input: root = []
Output: []

Example 3:

Input: root = [1]
Output: [1]

Program link: https://leetcode.com/problems/binary-tree-postorder-traversal/
