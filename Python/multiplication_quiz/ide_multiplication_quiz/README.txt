=Multiplication Quiz=

A program which tests the user on one digit mutiplication. 10 questions in total are asked.
The time limit for each question is 8 seconds. The user has 3 tries to get the right answer.
If the user does not answer the question in time or the user does not get the right answer in 3 tries,
the problem will be marked as incorrect. The user's score will be displayed at the end of the quiz.

<How to play>
1. Press enter to start the quiz.

2. The question will appear. Enter your answer and press enter to answer the question.

    - There is a 1 second interval per question.
