=Text Tool=

A program that handles text files and can conduct the following operations on them:
1) Searches the text file for a specific word or according to a regular expression
inputted by the user. The program will display the lines on which the word was found.
2) Replaces a specific word in the text file with a word inputted by the user.
3) Displays the word count for the top ten appearing words in the text file and
a word of choice inputted by the user.
