=Text Analyzer=

A program that provides quick statistics about the contents of a text file.
The program can display the following information about a text file:
- Content of the text file.
- Word count
- Character count
- Line count
- Top ten words appearing in the text file
- The count of a specific word.

<Usage>
Use: [option] [.txt file path] (Ex. "ruby textanalyzer.rb -d text.txt")

Options:
[-d]: Display text.
[-wc]: Display word count.
[-c]: Display character count with spaces.
[-l]: Display total line count.
[-wt]: Displays the count of top ten most appearing words.
[-sc] [word]: Displays the number times 'word' appeared within the text
